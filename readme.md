# React UseState

A collection of projects focused on mastering the `useState` hook in React.

**Repo:** https://github.com/iamrkesharwani/react-use-state

## Projects

### 1. Counter App

- **Features:** Increment, decrement, and reset functionality.
- **Logic:** Decrement button disables at zero.

### 2. Text Mirror

- **Features:** Real-time text mirroring and character counting.
- **Logic:** Includes a clear text action.

### 3. Theme Toggle

- **Features:** Light/Dark mode switcher.
- **Logic:** Dynamic style updates and smooth transitions.

### 4. Todo List

- **Features:** Add, complete, and delete tasks with unique IDs.
- **Logic:** Tracks total/completed counts and clears finished items.

### 5. Temperature Converter

- **Features:** Real-time Celsius/Fahrenheit conversion.
- **Logic:** Support for swapping conversion directions.

## Setup

1. Clone the repo.
2. Run `npm install`.
3. Run `npm start`.
